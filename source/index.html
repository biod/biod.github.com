Introduction

BioD is an open source software library for analysis of biological sequence data written using the D programming language. The library aims at developing efficient, low memory bioinformatics applications used for high throughput biological data.

Obtaining BioD

Clone the repository ``` $ git clone https://github.com/biod/BioD.git ```

Building

BioD can be built using the make file as well as using the meson build.

Installing using the DUB package manager

List BioD as a dependency on your dependencies list
