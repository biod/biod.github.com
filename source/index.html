BioD

BioD is a fast and memory efficient bioinformatics library written in the D programming language whose aim is to:

  • Provide a platform for developing high-performance computational biology applications using the D programming language through automatic parallelization of tasks where possible
  • Avoiding unnecessary memory allocations
